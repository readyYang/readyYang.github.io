hint：网管小张审计日志时发现疑似黑客使用Bash漏洞对公司的路由器进行了攻击，小张通过技术手段获取到了路由器固件dump文件，但是小张不懂逆向分析，请帮助小张分析出后门程序所使用的远程服务器和端口.(flag文件在/home/ctf/flag.txt文件中，Flag形式为 md5(网址：端口)}，注意网址不包含http比如：md5(www.baidu.com:80)=5411a465b86d593986b8b9f9b1c1788c) 注意：得到的 flag 请包上 flag{} 提交

网管还要管这种事的吗？这道题我很早就开始试了，也是做的真的挺久的，但我自己什么都没有看出来，看了别人的wp才勉强知道怎么做。

第一次做固件分析，固件逆向是指在不对嵌入式系统进行实际运行的情况下，通过对固件文件进行逆向解析，分析固件中各代码模块的调用关系及代码内容，从而发现嵌入式系统中可能存在的漏洞及后门的一种技术手段。现在我觉的靠自己是不可能做出这道题的，因为我看师傅们的wp也是看了很久才看懂一些。网上搜到的一些有关固件逆向的文章也是看不懂一点，好像固件逆向和工控还有物联网有点关系。

这道题的固件是路由器固件，给的是一个.bin文件，我自己做的时候是想把.bin文件转换成别的类型的文件，然后再去逆向分析它，结果当然是毫无进展。

这道题首先要用到binwalk，这是一个固件分析工具，它可以扫描固件映像以查找许多不同的嵌入式文件类型和文件系统。先在linux虚拟机上安装好binwalk，用binwalk -e 文件地址提取bin文件也就是固件映像中找到的所有文件，提取出来的文件会放在映像的同级文件夹下，打开生成的文件夹

里面有四个文件，第一个文件夹是空的，第三个我会解压缩失败，不知道为啥，但这些其实不重要，主要的是第四个文件。路由器文件系统这个文章是大神对路由器固件的分析，里面讲的很详细了，看完对这道题的理解帮助很大。

按照网上wp里大佬的步骤接下来就是安装firmware-mod-kit，(Firmware Mod Kit是一种工具，它允许轻松地对各种嵌入式设备的固件映像进行解构和重构。它主要针对基于Linux的路由器，但应该与使用常见固件格式和文件系统（如TRX/uImage和SquashFS/CramFS）的大多数固件兼容。它的功能包括固件文件的解包和打包、固件提取文件系统的解压和压缩、DD-WRT Web Pages的修改等)。但我按他们的步骤来的时候每一步都出现了问题，这害我差点放弃这道题，接下来我会把我的解决方法加上

sudo yum install git build-essential zlib1g-dev liblzma-dev python-magic

这步是安装安装firmware-mod-kit时所依赖软件包，我在这步出现以下报错

这里是liblzma-dev安装失败了，好像是因为不支持新版ubuntu。所以需要在前面加上命令sudo apt-get install liblzma5=5.2.4-1ubuntu1.来回退版本

这三步分别是克隆firmware-mod-kit的github仓库内容，移动到指定文件夹和编译可执行文件。我在克隆仓库时一直报错，这是实际上网络问题，找个好点的时间段或者挂个代理多试几次就好了。随后我又在编译时报了很多莫名其妙的错，如果无视掉，接下来的步骤同样进行不了。firmware-mod-kit Ubuntu22.04编译通过，在这篇文章里介绍了编译失败的原因，主要是软件太老了，文章末尾还给了一个懒人包地址,好人啊

换懒人包重新来，克隆库同样要多试几次。编译过程等一会就编译好了，将120200.squashfs放入firmware-mod-kit-improveVer目录下然后运行命令./unsquashfs_all.sh 120200.squashfs然后就可以在squashfs-root的tmp文件夹下找到名为backdoor的文件，这步有的wp作者有多做了几步，但是结果好像是一样的，我就没有去深入了解而是选择最简洁的方法

unsquashfs_all.sh是 firmware-mod-kit 工具包中的一个脚本，它用于提取 SquashFS 文件系统。SquashFS 是一种常用于嵌入式设备和只读文件系统的压缩文件系统。

unsquashfs_all.sh脚本会尝试使用多种不同的工具来提取 SquashFS 文件系统，以支持不同版本的 SquashFS 文件系统。当你运行这个脚本时，它会自动检测 SquashFS 文件系统的版本，并使用合适的工具来提取文件系统

照例，分析程序：32位带upx壳

脱壳，放入32位IDA中，查看字符串找到符合要求的字符串

跟进字符串，有两处引用，一处没用，一处可以找到端口

所以flag是echo.byethost51.com:36667的MD5加密33a422c45d551ac6e4756f59812a954b
